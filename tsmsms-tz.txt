Техническое задание. Разработка сервиса Tsmsms

1. Есть начальная реализация сервиса Tsmsms (https://github.com/antoncom/tsmsms)
2. Основные функции, возлагаемые на данный сервис:
    2.1. Чтение SMS из очереди входящих
    2.2. Отправка SMS
    2.3. Предоставление полученных SMS на шине UBUS
    2.4. Отправка SMS-сообщения путём вызова метода на UBUS
3. Должна быть предусмотрена возможность повторной отправки SMS в случае ошибки.
4. В случае, когда количество попыток отправить SMS истекло, текст SMS должен быть передан сервису отправки Email. При этом, факт ошибки отправки SMS должен быть отражен в журнале событий и в Web UI.
5. Должен быть предусмотрен режим отладки сервиса.
6. Сервис должен быть реализован, используя метод конечных автоматов. Диаграмма состояний сервиса прилагается.
